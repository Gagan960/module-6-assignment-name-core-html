1. <html>: The root element that wraps all the content of the HTML document.
  2. <head>: Contains meta-information about the HTML document like title, charset, and links to styles/scripts.
  3. <title>: Sets the title of the web page shown in the browser tab.
  4. <body>: Contains all the visible content of the web page such as text, images, links, etc.
  5. <a>: Creates hyperlinks to other pages or sections.
  6. <img>: Displays images on the web page.
  7. <p>: Defines a paragraph of text.
  8. <h1> to <h6>: Define headings, where <h1> is the most important and <h6> is the least.
